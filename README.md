# Native DB

### 这是什么？
Native DB 库是为Mod开发者准备的一个数据库，你可以便捷的在里面搜索、查询 并使用 游戏指令. 

并且我们将所有已知数据进行的汉化，开发者可便捷的搜索、想要的内容.

数据库添加了高效的搜索功能，可以快速的在 Native DB 库中进行搜索你想要的内容.

很久很久以前挖的一个坑, 现在终于填上了。
